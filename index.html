Match the Flight Number



Instructions


  • You will be given a random flight number at the start of the game

  • There are 4 planes below. By clicking on a plane you will add a specific amount to your flight number

  • Win by matching your flight number to today's flight number. You lose the game if your flight number goes above today's flight number

  • The value of each plane below is hidden until you click on it.

  • The value of each plane changes every time the game restarts
